# Quick Percentage Calculator

A lightweight browser extension for quickly calculating percentage discounts and percentage differences.

## Built With
- JavaScript
- HTML
- CSS

## Why I Built This
While working in ecommerce promotions I frequently needed to quickly check discount percentages and price differences. This extension allows those calculations to be done instantly without leaving the browser.

## Features
- Calculate percentage discounts
- Calculate percentage difference between values
- Simple popup UI
- Runs locally in the browser
